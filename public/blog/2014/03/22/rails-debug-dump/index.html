Rails Debug Dump

Posted on Mar 22, 2014

For displaying debug dump in Rails on development environment, edit application.html.erb. Add this lines inside the <body> tag.

<body>
  <%= yield %>
  <%= debug(params) if Rails.env.development? %>
</body>

To make things look nicer add some SCSS.

@mixin box_sizing {
 -moz-box-sizing: border-box;
 -webkit-box-sizing: border-box;
 box-sizing: border-box;
}

.debug_dump {
  clear: both;
  float: left;
  width: 100%;
  margin-top: 45px;
  @include box_sizing;
}
